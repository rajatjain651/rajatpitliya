Automated Gmail Creation

This is a project that I did in march - april of 2021 don't remember the exact month. This project was part of a assignment during campus placements.

In this short project I created a gmail account using selenium web driver and other third party python packages. You Can find the live video of it on Youtube here

So the main goal was to create a gmail account with complete automation basically not doing anything manually. Although I did said the otp to the microphone but it couldn't be avoided ( to deal with fake accounts in my opinion).

Here I used the 3rd party python packages like pyaudio and speech recognition to deal with audio inputs and then for conversion to text to be used by selenium web driver later.

Technologies:
  • - Python
  • - Selenium
  • - Pyaudio
  • - Speech Recognition
  • - Regular Expressions
