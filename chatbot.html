Django/Flask Chatbot Webapp

So this is a project that I worked on as an intern at Osswal Infosystem.

In this project the target was to create a chatbot for the clients of sap b1 software where they can ask questions like who is my best salesman or what is my sales data for a given interval of time for a particular salesman etc.

In this project I have used various regular expression techniques to extract information from the input data/text and get the required answers from that extracted info.

I have used the third party python library pyodbc to interact ( basically creating connection with database and the python) with the sql server database using python.

Here I have used flask web framework to run it on a local server.

My future plan with this project is to have a audio input and audio ouput to make it more seamless.

Technologies:
  • - Python
  • - Flask Web Framework
  • - SQL Server
  • - Regular Expressions
  • - Pyodbc
